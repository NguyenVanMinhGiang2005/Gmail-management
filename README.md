## Getting Started

### Prerequisites

- Node.js 24+ or Bun
- Python 3.14.4

### Frontend Setup

Using npm:

```bash
cd frontend
npm install
npm run dev
```

Using Bun:

```bash
cd frontend
bun install
bun run dev
```

The frontend will be available at `http://127.0.0.1:8080`
